
## API_CRUD_SIMPLES

Esta API foi de desenvolvida com intuito de mostrar, de forma simples, uma API que faz cadastro de usuarios com um CRUD simpes(crir, listar, atualizar e deletar).



### ferramentas utilizadas
* node e express
* prisma : para o gerenciamnto do banco de dados
* mongoDB : Para o banco de dados
* npm como gerenciador de pacote


## Comandos para utilizar este projeto:

 Para baixar as dependencias:
 ````
 npm install
 ````

 Para baixar rodar o servidor:
 ````
 npm start
 ````
